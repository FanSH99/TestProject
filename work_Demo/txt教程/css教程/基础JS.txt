
js基础

    JavaScript 是web开发领域中，功能强大的编程语言。（脚本语言），
    不需要配置环境，在浏览器上可以直接运行的解释性语言。
    JAVA需要配置环境。


    HTML5、CSS3、JavaScript关系
        HTML 结构，从语义化的角度，描述页面结构。
        CSS 样式，从审美的角度美化页面。
        JavaScript 行为。 从交互的角度，来提升用户体验。 CSS3可以做一部分但没有JavaScript方便。


    JavaScript的书写位置或引用方式
        1.写在行内   ----行内式
            <input type="button" value="按钮" onclick="alert('Hello World')" />
        2.写在script标签中  ---内嵌式
            <head>
            <script>
                alert('Hello World!');
            </script>
            </head>
        3.写在外部js文件中，在页面引入  ---外链式
            <script src="main.js"></script>

    js变量声明：
        1.声明变量
            var 变量名;
        2.为变量赋值
            变量名=值;
        3.声明变量是直接赋值
            var 变量名=值;
                ex：
                var uname="张无忌";
                var age=20;
        注意：
        1.允许在一条语句中声明多个变量，用逗号隔开变量名。
            var uname="韩梅梅",uage=20;
        2.如果声明变量，但未赋值，则值默认为undefined
        3.声明变量时可以不适用var,但不推荐
            uname="tom";


    js变量名命名规范：
        1.不允许以数字开头
        2.不允许使用关键词和保留关键字
        3.最好见名知意
            var uname; var uage;
        4.允许包含字母，数字，下划线(_),$
            var $name="Tom";
        5.尽量使用小驼峰命名法
            var userName;
            var uname;
            var _uname;//下划线
            var user_name;//下划线
            var UserName;//大驼峰命名法

    js变量的使用：
        1.声明变量未赋值，值默认为undefined
        2.使用未声明过的变量 报错
        3.赋值操作
            变量名出现在=的左边，一律是赋值操作
                var uname="林妹妹";
        4.取值操作
            变量只要没出现在=的左边，一律是取值操作
                var uage=30;
                console.log(uage);
                var num1=uage;

    js 函数的声明和调用
        普通函数的声明和调用（无参数无返回值）
            1.声明
                function 函数名(){
            //函数体
             若干可执行的语句
  
            }
            2.调用函数
                在任意javascript合法的位置处通过 函数名(); 对函数进行调用。
 

        带参函数的声明和调用
            1.声明
                 function 函数名(参数列表){
                //函数体
                }
                参数列表：可以声明0或多个参数，多个参数间用逗号隔开
                声明函数时，声明的参数，称之为“形参”
            2.调用
                函数名(参数值列表);
                注意：调用函数时，传递的参数数值，称之为“实参”。
                    尽量按照声明函数的格式进行调用
        
        带返回值函数声明和调用
            1.声明
                function 函数名(参数){
                //函数体
                return 值;
                //return关键字，程序碰到return关键词，就立马跳出并且把值带出去
                }                    
                注意：最多只能返回一个值
            2.调用
                允许使用一个变量接收函数的返回值
                var result=函数名(实参);


    JS中的作用域分两种：
        1. 函数作用域
            又称为局部变量，只在声明的函数中有效。在某个function中声明的函数。
            ex：
            function test(){
            var num=10;
            } 
        2.全局作用域
            又称全局变量，一经声明，任何位置都能用。在最外层(<script>中)定义的函数

            1.不在function中声明的变量，为全局变量
            2.声明变量不使用var，无论任何位置声明，都是全局变量（不推荐）
            注意：
                全局变量和局部变量冲突时，优先使用局部变量。
            3.变量的声明提前
        
        什么是声明提前
            在JS程序正式执行之前，function声明的函数，
            会将所有var声明的变量，都预读（声明）到所在作用域的顶部，但赋值还是保留在原位。
