
input 标签可以用的属性：

<input> 标签规定了用户可以在其中输入数据的输入字段。
<input> 元素在 <form> 元素中使用，用来声明允许用户输入数据的 input 控件。
输入字段可通过多种方式改变，取决于 type 属性。
<input> 元素是空的,它只包含标签属性。
提示: 你可以使用 <label> 元素来定义 <input> 元素的标注。

	onfocus定义输入时样式
	在元素获得焦点时被触发，即：获得焦点事件；常用于 <input>、<select> 以及 <a>标签中。
	 注：onfocus 属性不适合用于以下元素中使用：
	<base>、<bdo>、<br>、<head>、<html>、<iframe>、<meta>、<param>、<script>、<style> 或 <title>。
	 onblur定义输入结束时样式
	onblur属性在元素失去焦点时触发，即：失去焦点事件；常用于表单验证代码（例如用户离开表单字段）中。
	  onmouseover 鼠标移入时定义样式
	onmouseout 鼠标移出时定义样式
