下载好微信小程序之后，用微信扫码，
新建一个目录，命名--之后选择js模板确定即可创建一个项目。

一个小程序包含一个描述app和多个描述各自页面的page
一个小程序主体由三个文件组成。
app.js写js代码
app.json全局配置文件
app.wxss公共样式--全局样式

一个小程序页面由四个文件组成。
js文件写页面逻辑。
wxml相当于html，写页面结构，
json写页面配置。
wxss相当于css，写当前页面样式表

```
<text>相当于span</text>
<view> 相当于div</view>
```
注意：json文件中必须双引号，结尾不能有逗号，而且不能加注释。

entryPagePath  小程序默认启动页面，不设置则为index
pages 页面路径表 ，如果想写多个页面
在pages里面添加页面后会自动添加，

window中的配置
// 导航栏背景颜色默认是白色  --必须用16进制的颜色,全局配置中所有页面都会变。
"navigationBarBackgroundColor": "#ee3f4d",
//导航栏字体颜色 默认是白色 还可是black黑色
"navigationBarTextStyle":"black"
  // 导航栏标题内容
 "navigationBarTitleText": "初始小程序",
"navigationStyle":"default" //---默认导航 一般都用这个
 "navigationStyle":"custom"// ---自定义导航，写上之后，所有的导航栏样式不生效，需要自己加类名然后在wxss文件中在自己写样式

"enablePullDownRefresh":true 是否开启下拉刷新
"backgroundColor":// ---下拉树新的背景色
backgroundTextStyle  下拉loading的样式，仅仅支持dark和light，默认是dark
backgroundColorTop  顶部窗口的背景色，仅ios支持
enablePullDownRefresh 是否开启下拉刷新
homeButton   在非首页或非tabber页面导航栏显示home键

tarBar中的配置：

    "selectedColor":"#ee3f4d", 选中时的字体颜色

    "borderStyle":"white",
